About doNOTEforget

LOCAL STORAGE

This app is storing your notes locally, in your browser’s cache on the device that you are using right now. You can access them any time as long as you use the same browser and device.

NO NEED TO LOGIN

Unlike Cookies, Local Storage data is not sent to a server. That means that you don’t need an account: your browser will keep your notes safe and up to date even after you close this website.

NEW BROWSER - NEW STORAGE

Due to storage terms, the notes that you have made using this Browser can not be accessed from another browser or device. But you can start a new Note-List on unlimited amount of Devices.

THERE IS MORE TO COME

This project is going to evolve with me :)

But for now, thank you for using “doNOTEforget”, hope you enjoy it, and have a great day!
